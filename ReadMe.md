
1. Create Virtual Environment

syntax: 'python -m venv <name>'

'python -m venv venv'

2. Activate Virtual Environment 

windows: 'venv\Scripts\activate'

3. Install the required libraries (numpy, pandas, openai, langchain, etc)

'pip install -r requirements.txt'

4. start your application

"python App.py'

## versioning the code

1. Check if already in a Git repository

'git status'

2. Initialize a new empty Git Repo

'git init'

3. Move file to Staging Area

'git add <file name>
OR
'git add .'

4. Commit the changes

'git commit -m "add commit message here"'

5. List all commit

'git log --all'
